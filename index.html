Boatless

A bloatless package based on Boilerplate containing normalize.css and Modernizr, styled using Stylus, and generated using Gulp (which runs on Node.js).

Less is more and what not. Instructions included.

Download Fork it

Not a framework

Sorry. Boatless is a just a different approach at getting started with development. This could not have been done without the awesome people behind HTML5 Boilerplate, Normalize.css, Modernizr, Node.js, Gulp, Stylus, and well the list is long but thank you.

Boatless has a folder structure which focuses on clear naming and structuring which is a somewhat different approach when comparing to the commonly used templates and frameworks. Normalize.css is combined with the default CSS from HTML5 Boilerplate and structured for minimal overriding CSS rules.

Please note that I am currently conducting browser tests and making minor tweaks to the setup.
